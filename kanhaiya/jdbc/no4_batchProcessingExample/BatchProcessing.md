## Batch processing in JDBC

- it used to perform operation in jdbc 
- it performs task one by one
- it reduce number of database call and increase efficiency 

## Stored procedures

 - stored procedure in sql is pre-compiled and reusable set of sql statement stored in database.
 - it can take input parameters, executes multiple sql statements, and return output parameters or result set.
 
 
## Callable statement

- callable statement are used to execute stored procedure in database.
-  and efficient way to interact with database procedures and encapsulate complex locic and improve performance

### Types of callable statements

1. IN parameter 
- it is used to pass an input to the procedure using setter methods

2. OUT parameter
- it is used to retrive values from the procedure using registered out parameter method

3. IN OUT
- it acts as both input and output
